Detailed Description

template<class TProc>
class kiwi::dsp::PerformCallBack< TProc >

Templated implementation of IPerformCallBack. Templated on type of processor.

The PerformCallBack acts as std::function but encapsulate a pointer of method to avoid overhead generated by an std::function indirection.
