template<typename TModel>
struct kiwi::model::Factory::isValidObject< TModel >

isValidObject type traits

To be considered as valid an object must:

  • not be abstract.
  • inherit from model::Object.
  • be constructible with flip::Default.
  • be constructible with a string and a vector of atoms.
