Parameters
textThe string to parse.
Returns
The vector of atoms.
Remarks
For example, the string : "foo \"bar 42" 1 2 3.14" will parsed into a vector of 5 atoms. The atom types will be determined automatically as 2 #Atom::Type::TAG atoms, 2 #Atom::Type::LONG atoms, and 1 #Atom::Type::DOUBLE atom.
