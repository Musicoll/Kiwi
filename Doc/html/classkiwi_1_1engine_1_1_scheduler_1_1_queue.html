template<class Clock = std::chrono::high_resolution_clock>
class kiwi::engine::Scheduler< Clock >::Queue

A class that holds a list of scheduled events.

Implementation countains a list of events sorted by execution time that is updated before processing using commands. A queue is created for each pair of producer/consumer.
